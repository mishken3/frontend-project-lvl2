## Description

This is training JS-project on [Hexlet](https://hexlet.io), app that compares two config files and shows difference

## Getting Started

### Setup:

```
git clone https://github.com/mishken3/frontend-project-lvl2
npm install
npm link
```

### Launch:

```
gendiff <file1.json> <file2.json>
```

### Available files:

```
json
yaml
ini
```

### Available formats:

change format by --format option. Available formats:

```
stylish
json
plain
```
